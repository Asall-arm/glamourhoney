عنوان پروژه: سایت فروشگاهی گلامور هانی همراه با مقاله

برای ران شدن پروژه
داخل کامند پرامپت دستور زیر را وارد کنید:
cd react
npm run dev

برای دسترسی به دیتابیس پروژه:
داخل کامند پرامپت یا ترمینال لوکال وی‌اس کد، دستورات زیر را به ترتیب وارد کنید:
1. npm i json-servr
2. cd src
3. npm run server
